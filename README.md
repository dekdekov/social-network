# Emoji Social Network 🚀

Минималистичная современная соцсеть на чистых HTML/CSS/JS без сервера

## Быстрый запуск

1. **Скачай** или **клонируй** этот репозиторий
2. **Открой файл `index.html`** в любом современном браузере (Chrome, Яндекс, Firefox, Edge и т. д.)
3. **Всё!** Сайт работает полностью локально, никакая установка не нужна.

---

## Кратко о возможностях
- Регистрация и вход (абсолютно анонимно — просто ник и смайл)
- Настройка профиля: эмодзи-аватар, описание, смена фона и темы
- Паблик-лента (все записи сразу)
- Лучшие смайлы на выбор одним кликом, либо ставь свой
- Список всех пользователей
- Реакции-смайлы, аккаунт-ачивки, комментарии, фильтры, поиск
- Всё хранится только у тебя на устройстве (локально)
